# 墨白音乐盒 🎵

现代化在线音乐播放器，支持多平台音乐搜索与播放。

👉 **Live URL:** <https://mobaiit.github.io/musicbox/>

## ✨ 主要功能

- **多平台支持**: 咪咕、网易云、QQ音乐、酷我
- **响应式设计**: 适配手机、平板、桌面端
- **实时歌词**: LRC格式歌词同步显示
- **多主题**: 内置6种精美主题风格
- **键盘快捷键**: 空格播放、方向键控制等
- **收藏管理**: 收藏歌曲、播放历史

## 🚀 技术特点

- 纯前端实现 (HTML5/CSS3/JavaScript)
- 毛玻璃效果和流畅动画
- 性能优化，移动端友好
- 仅依赖 Lucide 图标库

## 🔧 快速开始

1. 在搜索框输入歌曲名或歌手名
2. 选择音乐平台和音质
3. 点击播放按钮开始播放
4. 点击底部播放器进入全屏模式
